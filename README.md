# HomeNest

## Installation

Install my-project with npm

1. **Clone the repository**:
    ```bash
    git clone https://github.com/username/project.git
    ```

2. **Navigate to the project directory**:
    ```bash
    cd project
    ```

3. **Install dependencies**:
    ```bash
    npm install
    ```

4. **Start the server**:
    ```bash
    npm run server
    ```

5. **Start the client**:
    In another terminal, run the following:
    ```bash
    npm run client
